Betriebspunkt: Niederdruck 2.2 bar, Hochdruck 13.8 bar, COP = 3.25

Was ist ein p-h-Diagramm?

Das Druck-Enthalpie-Diagramm ist das wichtigste Werkzeug in der Kälte- und Klimatechnik:

  • Y-Achse: Druck p in bar (logarithmisch skaliert)
  • X-Achse: Spezifische Enthalpie h in kJ/kg
  • Sättigungskurve (Glocke): Trennt flüssige Phase von Dampfphase
  • Nassdampfgebiet: Bereich innerhalb der Glocke (Gemisch aus Flüssigkeit + Dampf)
  • Isobaren: Horizontale Linien = konstanter Druck
  • Isothermen: Kurven = konstante Temperatur

COP-Berechnung aus dem Diagramm

Enthalpien ablesen:

  • h₁ (nach Verdampfer) ≈ 407 kJ/kg
  • h₂ (nach Kompressor) ≈ 435 kJ/kg
  • h₃ (nach Verflüssiger) ≈ 268 kJ/kg
  • h₄ (nach Expansion) ≈ 268 kJ/kg (isenthalp!)

Berechnungen:

Kälteleistung Q₀: h₁ - h₄ = 407 - 268 = 139 kJ/kg

Kompressorarbeit W: h₂ - h₁ = 435 - 407 = 28 kJ/kg

Heizleistung Q: h₂ - h₃ = 435 - 268 = 167 kJ/kg

COP = Q / W = 167 / 28 = 5.96 (theoretisch)

Real gemessen: COP = 3.25

Die Differenz kommt von:
• Wärmeverlusten an Gehäusen
• Druckverlusten in Leitungen
• Motorwirkungsgrad Kompressor
• Ölrückführung und Nebenverbrauchern

Optimierungspotential im p-h-Diagramm sichtbar

⚠️ Problem: Punkt ① liegt zu weit rechts

Aktuelle Überhitzung: 12.7°C

Im Diagramm: Punkt ① hat großen horizontalen Abstand zur Taulinie (rechter Ast der Sättigungskurve)

Folgen:

  • ❌ Zu viel Verdampferfläche wird für Überhitzung "verschwendet"
  • ❌ Zu wenig Fläche für die eigentliche Verdampfung (Phasenwechsel)
  • ❌ COP-Verlust von ca. 5-10%

Lösung: EEV weiter öffnen

  • ✓ Mehr Kältemittel-Durchsatz
  • ✓ Punkt ① wandert nach links (näher an Taulinie)
  • ✓ Optimale Überhitzung: 5-8°C
  • ✓ Mehr Verdampfungsfläche = höherer COP

Erwartung: COP 3.25 → 3.35-3.40 (+3-5%)

  • ✓ Druckverhältnis: 13.8 / 2.2 = 6.3 → moderat (gut für Kompressor)
  • ✓ Temperaturhub: 50°C - (-10°C) = 60 K → für Grundwasser-WP sehr gut
